# Plant images

Upload plant equipment photos into this folder.

When the images are uploaded, let Claude know — the file names can be
anything; Claude will update the `<img>` references in `plant-hire.html`
to match whatever you upload (galleries and the page hero banner).

Suggested naming (lowercase, hyphenated) if helpful:
  excavator-*.jpg    forklift-*.jpg    dumper-*.jpg
  generator-*.jpg    roller-*.jpg      skid-steer-*.jpg
